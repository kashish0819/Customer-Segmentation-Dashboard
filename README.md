# Customer Segmentation Dashboard

## Project Overview

Customer Segmentation is a Machine Learning project designed to identify distinct customer groups based on demographic, financial, and behavioral characteristics.

The project uses clustering algorithms to discover meaningful customer segments and provides actionable marketing insights through an interactive Streamlit dashboard.

---

## Features

### Data Analysis

* Missing Value Analysis

* Feature Engineering
* Data Preprocessing

### Exploratory Data Analysis

* Customer Demographics Analysis
* Income Distribution

### Machine Learning

* Principal Component Analysis (PCA)
* K-Means Clustering
* Hierarchical Clustering

### Customer Intelligence

* Customer Personas

* Marketing Recommendations

### Interactive Dashboard

* Executive Summary

## Dataset

Customer Segmentation Dataset from Kaggle.

Dataset contains customer demographic, financial, and behavioral information including:

* Age
* Gender
* Income Level
* Coverage Amount
* Premium Amount
* Policy Type
* Customer Preferences

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Customer-Segmentation-Dashboard.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run dashboard:

```bash
cd dashboard
python -m streamlit run app.py
```
